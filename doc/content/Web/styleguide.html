Page Dimensions
Position
The layout will remain in the horizontal center of the window.

Width
The page will maintain a width of 720 pixels.

Height
The page will maintain a height of 100%. There will consistently be a 20 pixel high border at both the top and bottom of the window with the background showing through.

Modules
Any smaller box within the larger site box is called a module. Whitespace between modules must be exactly 10 pixels on all sides.



Header
The header should remain simple and clean, containing only the logo on the left with the blue-striped background.

Navigation
Navigation will always be placed below the header. Navigation is made up of rollover images. Current page navigation button will turn green to indicate current location.

Sub-Navigation
Sub-navigation will always be placed below the navigation. Sub-navigation links are text. Links are 75% gray while the current page is not a link and is black and bold. Hovering over the available links will turn those items to black.




Content
All content pages shall contain a section header, subsection header and body text.




Colors
These colors are provided for use in accent graphics and accompanying materials.


Photos / Images
Photos / Images will be surrounding by a two pixel thin black border (inside).
